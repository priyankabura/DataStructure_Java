Following are the java files included-

SkipListDriver.java - driver program to read the input file and perform the given operations and output the result and time taken.

SkipListImpl.java - holds the implementation of skip list and various functions like first,last,add,remove,findIndex,floor,ceiling,contains,rebuild,isEmpty etc.

SkipList.java - Interface whose functions are implemented by SkipListImpl.java

TreemapTest.java - program to perform the add,remove and contains operations on the given input using java’s inbuilt TreeMap data structure.

————————————————————————————————————

Steps to Run SkipListImpl.java:

Compilation - javac SkipListImpl.java
Execution - java SkipListImpl input_filename


Steps to Run TreemapTest.java:

Compilation - javac TreemapTest.java
Execution - java TreemapTest input_filename
————————————————————————————————————

input_filename(eg: a9.txt/b8.txt)
